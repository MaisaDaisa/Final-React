# Pokemon Team Builder

This is my work for the final project of React course of BTU, I have used the PokeAPI many times already and decided to redo one of the old projects i have done on vanilla javascript with purely the team building part enhanced. I am sorry, i could have done more or better in some prats, but due to the short deadline, my job and exams, i could not allocate more time to this project. please keep in mind that i have worked with frontend for over 2 years and i am very familiar with react.

## Tech Stack

- **Framework**: React
- **Routing**: React Router
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Data Source**: PokeAPI

## Installation

1. **Clone the repository**

    ```bash
    git clone https://github.com/MaisaDaisa/Final-React
    ```

2. **Install dependencies**

    ```bash
    npm install
    ```

3. **Start the development server**

    ```bash
    npm run dev
    ```
